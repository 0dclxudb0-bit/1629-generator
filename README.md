# 1629 Username Generator

A sleek Y2K-themed Discord username generator with a chrome gradient aesthetic.

## Features

- **Multiple Generation Modes**:
  - 3-letter random usernames
  - 4-letter random usernames
  - OG mode with curated aesthetic words

- **Customizable Output**: Generate 1-500 usernames at once

- **Easy Copy**: One-click copy to clipboard

- **Modern Design**: Dark theme with animated chrome loading screen

## Live Demo

Visit [1629-generator](https://0dclxudb0-bit.github.io/1629-generator/) to use the generator.

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/0dclxudb0-bit/1629-generator.git
cd 1629-generator
```

2. Open `index.html` in your browser or serve it with a local server:
```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if http-server is installed)
npx http-server
```

3. Navigate to `http://localhost:8000` and start generating!

## How to Use

1. Select your desired username mode (3-letter, 4-letter, or OG)
2. Enter the amount of usernames to generate (1-500)
3. Click "Generate" to create the usernames
4. Click "Copy" to copy all generated names to clipboard
